CountDownLatch adn CyclicBarrier

发表于 2019-05-28 | 分类于 技术 | 阅读次数:

1.CountDownLatch

CountDownLatch latch = new CountDownLatch(7);
CountDownLatch可以用来计数,之前多线程读取日志数据时,在每个线程里latch.countDown();
所有线程提交后latch.await();
这里会等CountDownLatch里的计数变0才会往后执行.

2.CyclicBarrier

CyclicBarrier是一个同步的辅助类，允许一组线程相互之间等待，达到一个共同点，再继续执行。

CyclicBarrier c=new CyclicBarrier( int parties,Runnable BarrierAction);
会在cyclicBarrier.await();执行 parties 次后运行 BarrierAction()方法.
也就是一组线程(parties个线程)都达到某个状态后会执行.

！！！—但是如果线程池大小不够，那么之前的线程都不会释放资源—！！！
