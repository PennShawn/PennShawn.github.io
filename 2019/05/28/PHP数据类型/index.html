####php支持8种原始类型,包括4种标量类型,即boolean,integer,float/double和string; 2种符合类型,array(数组)和object(对象); 2种特殊类型,即resource(资源)和null.

1. 布尔型(boolean)

只有true或false,在一些特殊情况下0、0.0、’0’、”0”、空白字符串(“”)、只声明没有赋值的数组等也被认为是false.

\$i=’0’ 输出 false!
$i=-1 输出 true!

$a='字符串' 或 $a="字符串"
两者的不同之处在于,双引号所包含的变量会被自动替换成实际数值.
